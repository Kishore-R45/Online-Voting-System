# 🗳️ Online Voting System using PHP

This is a **web-based Online Voting System** developed with **PHP, MySQL, HTML, CSS, and JavaScript**. It streamlines the election process by allowing users to cast votes online and view real-time results, while admins can manage the election securely.

## 📌 Features

- 🔐 **Email-Based Voter Authentication**  
  Replaces photo-based identification with email login for enhanced accessibility and privacy.

- 🗳️ **One Person, One Vote**  
  Each user can vote only once per election to maintain fairness.

- 🧑‍💼 **Admin Dashboard**  
  Admins can add or remove candidates, monitor votes, and control the election session.

- 📊 **Live Vote Result Bar Chart**  
  Vote counts are displayed visually in a bar chart using custom styling.

- 💽 **MySQL Database Integration**  
  Stores voter data, candidate details, and vote logs securely via JDBC (PHP–MySQL).

- 🎨 **Custom Modern UI**  
  Responsive design using clean CSS (Bootstrap replaced with custom styles and animations).

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Charts:** HTML5/Canvas or Chart.js (as applicable)

## 🚀 How to Run (For Beginners)

1. **Install XAMPP or any local server**
2. **Clone this repo:**  
   ```bash
   git clone https://github.com/yourusername/online-voting-system-php.git
   ```
3. **Move project folder into htdocs (XAMPP)**

4. **Start Apache & MySQL from XAMPP control panel**

5. **Import database:**
   Open phpMyAdmin → Import database.sql from the project

6. **Open in browser:**
   Visit http://localhost/Online-Voting-System-master/
   Login/Register as voter or admin to test the full workflow
